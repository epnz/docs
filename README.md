# EPNZ Docs 项目介绍

## 项目简介

EPNZ Docs 用于展示和分享 EPNZ 的相关文档和资源。

## 功能特点

- 项目包含了一些 EPNZ 的文档和教程，用户可以通过浏览文档来了解 EPNZ 的功能和用法。
- 支持多语言文档，包括中文和英文。
- 支持文档搜索和导航，使用了algolia搜索，用户可以通过搜索关键字来快速找到需要的文档。
- 支持文档目录和侧边栏。

## 安装和部署

1. 克隆项目到本地
EPNZ Docs 是一个基于 VuePress 的文档项目，用于展示 EPNZ 的文档和教程。它使用了 VuePress 的默认主题，并添加了一些自定义的样式和插件。

2. 安装依赖
在项目根目录下运行以下命令来安装依赖：

```bash
npm install
```

3. 启动开发服务器
运行以下命令来启动开发服务器：

```bash
npm run dev
```

访问 [http://localhost:5173](http://localhost:5173) 查看文档站点

4. 构建静态文件
运行以下命令来构建静态文件：

```bash
npm run build
```

5. 署到服务器
将构建后的静态文件部署到服务器上，可以通过以下命令来生成静态文件：

```bash
npm run build
```
